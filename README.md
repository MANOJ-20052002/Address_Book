\# Address Book 📒



\## Overview

A C program that manages contacts - add, search, 

update and delete contacts stored in a text file.



\## Features

\- Add new contacts

\- Search contacts

\- Update contacts

\- Delete contacts

\- Save contacts to file

\- Load contacts from file



\## Files

| File | Description |

|------|-------------|

| main.c | Main program |

| contact.c | Contact operations |

| contact.h | Contact header |

| file.c | File operations |

| file.h | File header |

| populate.c | Data population |

| populate.h | Populate header |

| contacts.txt | Contacts database |
